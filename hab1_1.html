천간합 : 음양합, 부부합, 유정지합
갑기(토) : 中正之合(마음넓고 존경)
을경(금) : 仁義之合(과감,강직,인의)
병신(수) : 威嚴之合(위엄,편굴,잔인,색)
정임(목) : 淫亂之合(감정적,색)
무게(화) : 無精之合(아름답지만 薄情)

  지지합 : 육합, 남녀합, 부부합
자축(토) : 克合(地, 일지丑+子=복)
인해(목) : 生合(春,破,일지寅+亥=복)
묘술(화) : 克合(夏,문명,일卯+戌=복)
진유(금) : 生合(秋,도화,일酉+辰=복)
신사(수) : 克合(冬,刑,破,일지申+辰)
오미(화) : 生合(天,일지未+午=복)

1.년+월 : 부자유친, 조업계승
2.년+일 : 배우자 화합, 원만한 가정
3.일+시 : 노후행복, 자식과 화합
4.년+시, 월+시 : 중요시 안함(약함)
